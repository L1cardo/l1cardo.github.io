射频模式

板载 ExpressLRS SPI 接收器的 AIO 飞控默认设置500Hz。

要调整它，你需要进入 Betaflight CLI 并使用以下命令：

set expresslrs_rate_index = [序号]
save

上面的[序号]替换成以下内容：

  • 500Hz = 0
  • 250Hz = 1
  • 150Hz = 2
  • 50Hz = 3
